# Samyak Ayurvedic Hospital Management System

A comprehensive web application for managing hospital operations, patient records, and appointments.

## Features

- **Patient Management:** Registration, profile updates, and secure login.
- **Doctor Dashboard:** View today's appointments, manage patient records, and issue prescriptions.
- **Appointment System:** Book, reschedule, and cancel appointments with automated reminders.
- **Inventory Management:** Track medicine stock with low-stock alerts.
- **Prescription System:** Generate PDF prescriptions and send them automatically via email.
- **Google Authentication:** Secure sign-in for patients.

## Project Structure

- `backend/`: Node.js/Express API, MongoDB models, and utility scripts.
- `frontend/`: Modern vanilla HTML, CSS, and JavaScript interface.

## Setup Instructions

### Backend

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure environment variables in `.env`.
4. Start the server:
   ```bash
   npm start
   ```

### Frontend

1. Open `frontend/index.html` using a local server (e.g., Live Server in VS Code).
